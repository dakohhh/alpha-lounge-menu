# Alpha Lounge Menu

TypeScript + React + Vite single-page menu for Alpha Lounge Bar & Restaurant.

## Scripts

- `npm run dev` — local dev server
- `npm run build` — production build to `dist/`
- `npm run preview` — preview the production build

## Project layout

```
src/
  App.tsx              Main page
  components/          UI pieces
  data/menu.ts         Menu sections and items
  lib/image-slot.ts    <image-slot> web component
  styles/alpha.css     Global styles
  theme/               Palettes, typography, theme application
```
